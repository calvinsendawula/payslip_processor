Memory Isolation Feature (v0.2.1)

This application uses the new memory isolation feature from qwen-payslip-processor v0.2.1.

Starting with strict isolation: each window is processed in a separate process with a fresh model instance.

Note: The package will automatically fall back to medium isolation if strict isolation is not supported on your platform or fails for any reason.
